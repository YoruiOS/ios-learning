# ios-starter-kit

A baseline SwiftUI app you can clone for new projects.

## Setup (Xcode)
1. Create a new iOS App project in this folder named `StarterKit`.
2. Turn on **Treat Warnings as Errors** and **Strict Concurrency Checking** (Complete).
3. Add a test target `StarterKitTests`.
4. Replace default files with the templates in `/templates/swiftui`.

## Definition of Done
- Two screens wired via NavigationStack.
- App icon + launch screen.
- Unit tests (for ViewModel) and a basic UI test.
- Accessibility pass: labels, traits, Dynamic Type.
